# Restaurant POS
## Report & Meeting Minutes file in `docs` folder
## Install
### `Install Clerkpage`
```
cd clerkpage
npm install
```
### `Install Clientpage`
```
cd clientpage
npm install
```
## Usage
### `Run server`
```
cd server
python server.py
```
### `Run Clerkpage`
```
cd clerkpage
npm start
```
### `Run Clientpage`
```
cd clientpage
npm start
```
